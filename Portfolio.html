The TOTP-Generator programm is a Time-based One-Time Password (TOTP) generator, similar to the ones used in two-factor authentication apps like Google Authenticator. It uses only standard libraries and follows the RFC 6238 specification to generate secure, time-based codes. The project solely serves for education purposes and runs on the console.
